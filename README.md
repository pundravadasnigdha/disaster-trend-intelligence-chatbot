# Disaster Trend Intelligence Chatbot

## Overview

The Disaster Trend Intelligence Chatbot is an AI-powered system that enables users to analyze historical disaster data using natural language queries. Instead of manually searching datasets, users can ask questions in plain English, and the chatbot retrieves relevant disaster records and trends.

The project combines data analysis with Large Language Models (LLMs) to provide an interactive and user-friendly way of exploring disaster information.

## Features

* Natural language query processing
* Historical disaster data analysis
* Disaster-wise and location-wise search
* Year-wise filtering
* Conversational chatbot interface
* AI-assisted response generation using an LLM
* Excel-based disaster dataset support

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Ollama (Qwen Model)
* OpenPyXL

## Dataset

The project uses a historical disaster dataset containing information such as:

* Disaster Type
* Disaster Subtype
* Country
* Location
* Start Date
* End Date
* Disaster Group
* Event Name

## Sample Queries

* Show flood events in Andhra Pradesh.
* Earthquake records in Gujarat.
* Cyclone disasters in India.
* Wildfire incidents between 2015 and 2020.
* Lightning events in Odisha.
* Show disasters that occurred in 2021.

## How to Run

1. Clone the repository.
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Install the required Python libraries.
4. Place the dataset in the correct directory.
5. Run all notebook cells.
6. Enter natural language queries to analyze disaster data.
